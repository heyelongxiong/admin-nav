javascript: 是一种弱类型的前端浏览器脚本语言 跟本就不算整正的语言，因为没有类、模块化、作用域之类的的概念，所有的类、模块化、作用域都是模拟出来的，如：数组去重 js特性：封装：就是可以写一些自定义的方法进行封装，写成公共的方法，或者写成私有的方法，再或者写成互相联系的方法 如：购物车功能(shopCar)，有添加功能，有cookie存储功能，删除功能、清除功能，购买功能，全选功能 多态: 继承:一切皆对象，有对象就有原型(prototype)，就是解决了代码的冗余，实现了只写一个函数就能在多处(其他函数内部)使用此方法， function App(name){ this.name = name; } var app = new App(); console.log(app.name("紫漠")) 抽象:抽象就是将一些操作的方法提取起来，对某一个对象进行操作 数组对象： 数组的索引永远是从前到后，从0开始； 数组也是一个对象，是引用护具类型，一般是用于存放数据的一个数组集合， 可以对数组里面对的数据进行增、删、改、查，数组的操作方法基本都是内置的， 也就是js已经封装好的方法。数组支持循环访问数据，可以减少代码的冗余性 push():给数组的末尾添加一个或者多个数据，原始数组会进行给变，且返回新 数组的长度,也返回length长度，和unshift相反 unshift():将数据添加到数组的首位，改变数组的长度，返回数组的length值； pop():删除数组的末尾项数据，改变原始数组，然后返回移除的项。 shift():删除数组的第一项数据，会给变原数组，返回被删除的项，如果数组 为空，则返回undefined。 slice():截取数组，可以接受两个参数，第一个参数就是截取此数组的开始位置 (开始索引)，第二个参数是截取当前数组的结束位置(结束索引)，如果没有传 第二个参数，则会截取当前数组索引开始位置后所有的项，不会改变原始数组 返回被截取的数组项；有第二个参数，则截取到当前数组结束索引位置的前一项 不包括当前索引项 join():就是将数组转化为字符串，并且进行拼接，只有一个参数，可选，默认的 分隔符是逗号(,),其他传参，传什么，分隔符就是什么； map():就是映射，ES5新出的方法，将数组可以循环出来,此方法接受三个参数，分别是当前每一项、 当前项的索引、需要操作的原数组；不会改变原数组，返回一个被改变之后的新数组 forEach():也是循环遍历数组，ES5新出的方法，此方法接收三个参数，分别是当前每一项、 当前项的索引、需要操作的原数组；不会改变原数组，没有返回值 sort(): 数组排序，默认的是从小到大的升序排列，不管每一项是数字还是字符串，则都会 转换成字符串进行比较，然后排序，从每一项的第一位数字或者字母开始对比，小的在前面 splice():删除、添加、替换，此方法接收三个参数，第一个参数是截取或者添加或者替换索引 的开始，第二个参数是需要截取元素的几项，第三个参数是替换的值； reverse():将数组反向排序，改变原数组，一般用于可视化较多的项目 indexOf():查找数组元素，字符串元素，不改变原始数组，找到 则返回当前元素所在的下标位置，如果没有找到则返回-1； lastIndexOf():查找数组元素，字符串元素，从后向前查找， 不改变原始数组，找到则返回当前元素所在的下标位置，如果 没有找到则返回-1；
